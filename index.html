오픈라이더 별도 서비스 서버 구축

Description.

오픈라이더 앱 서비스에서 기본적인 API 이외에도 별도 목적에 의해 구축된 서비스 서버가 있습니다.
그중에서도 부하분산처리가 필요한 몇몇 서비스들을 개발하였습니다.

상세내용

  • 푸시 메세지 발송 서버
  • MQTT 메세지 프로토콜을 활용한 메세징 서버
  • 내부 데이터 분석을 위한 데이터 분석 프로젝트
  • 반복적인 배치 프로세스 실행을 위한 배치 서버
  • OPEN API 서비스를 위한 환경 구축

    • 기술스택

    Java, Spring, Vertx, Redis, Eureka, API Gateway(Gravitee), Elasticsearch, FCM

    오픈라이더 서비스 API 개발

    Description.

    오픈라이더 앱 서비스에서 사용하고있는 REST API 신규 개발 및 유지보수를 하고,
    기존 레거시 코드를 리팩토링하는 작업을 진행하고 있습니다.

    상세내용

  • 앱서비스에서 사용되는 대부분의 인터페이스 구현
  • 기존 Callback 함수 구조에서 async 함수로 변경

    • 기술스택

    Node.js, Express, Javascript

    오픈라이더 웹 서비스

    Description.

    오픈라이더 앱 내에서 서비스하고 있는 몇몇 웹뷰와 별도의 웹 프로젝트를 작성하였습니다.
    사용자에게 노출되는 웹 페이지 이외에도 내부 업무를 위한 페이지도 구성하였습니다.

    상세내용

  • 웹뷰를 통해 단말에서 제공하는 일부 서비스를 웹페이지로 제공
  • 앱서비스 데이터와 연동되는 오픈라이더 홈페이지 구성
  • 내부 CS 업무를 위한 사내전용 서비스 구성

    • 기술스택

    Node.js, Vue, Javascript, ORM(Sequelize), Webpack, Nuxt
